# Machine Learning USA Housing Price Prediction

## Problem Statement
The goal of this project is to predict house prices in the USA using machine learning techniques based on housing features.

## Dataset
USA Housing dataset containing:
- Avg. Area Income
- Avg. Area House Age
- Avg. Area Number of Rooms
- Avg. Area Number of Bedrooms
- Area Population

Target Variable:
- Price

## Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Selection
4. Model Building
5. Model Evaluation

## Model Used
- Linear Regression

## Evaluation Metrics
- R² Score
- Mean Absolute Error
- Mean Squared Error

## Insights
- Area income strongly affects house prices
- Number of rooms increases price
- Population has moderate impact

## Conclusion
The Linear Regression model successfully predicts housing prices with good accuracy.
